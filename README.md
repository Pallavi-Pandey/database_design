# Database Design Notes

A collection of database architectural designs and modeling notes for various applications.

## Project Content

- **System Designs**: Architectural diagrams and notes for:
  - `chat_app_design/`: Real-time messaging database schema.
  - `facebook_design/`: Social network entity-relationship diagrams.
  - `harry_porter/`: Specialized thematic data modeling examples.
- **Visual Assets**: `image-X.png` files illustrating schemas, tables, and relationships.
- `notes.md`: Theoretical documentation on normalization, indexing, and modeling best practices.

## Purpose
This repository serves as a portfolio of database architecture skills, showcasing high-level system design for complex, scalable applications.
